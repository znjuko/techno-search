How to run skill service :

to setup default user:
https://stackoverflow.com/questions/64166492/how-to-setup-an-admin-account-for-clickhouse

used env:
THREADS=2;CLICK_HOST=localhost;CLICK_PORT=9000;CLICK_USER=default;PORT=8089

to start docker clickhouse server:
docker run -d --name clickhouse-server -p 9000:9000 --ulimit nofile=262144:262144 yandex/clickhouse-server

to start docker clickhouse client:
docker run -it --rm --link clickhouse-server:clickhouse-server yandex/clickhouse-client --host clickhouse-server

to connect to clickhouse server as bash:
docker exec -it clickhouse-server bash
